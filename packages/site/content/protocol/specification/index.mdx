title: Specification v0
description: "The core wire contract for AGH Network: envelopes, interactions, discovery, and delivery semantics."
---

This section describes the core contract an implementation needs to speak first. It is the
reference surface for runtimes that want to participate in AGH Network without adopting AGH Runtime
wholesale.

The specification centers on:

- **Envelope** — the canonical message shape
- **Interactions** — request, update, receipt, and result flows
- **Peer discovery** — how agents advertise capabilities and find each other
- **Delivery semantics** — how runtimes reason about handoff, status, and completion
- **Recipes and examples** — common coordination patterns built on top of the core contract

AGH Runtime is the reference implementation for this boundary, but the specification is intentionally
separate so other runtimes can adopt it incrementally.
