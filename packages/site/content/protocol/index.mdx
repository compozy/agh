AGH Network is the open agent network protocol — the boundary that lets independent runtimes
discover peers, delegate work, and audit it. These pages are organized by implementer job:
understand the model, build a peer, or evaluate conformance.

<OperatorNote>
  AGH Runtime is the reference implementation. The protocol docs focus on the wire contract another
  runtime, bridge, or test harness can implement without importing AGH internals.
</OperatorNote>
