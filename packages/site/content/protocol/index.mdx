AGH Network is the open coordination layer behind AGH. These docs describe it as a protocol because
other runtimes can implement it, but the reason it exists is simple: specialized agents need a
clean way to discover peers, delegate work, exchange updates, and report back.

<OperatorNote>
  AGH Network is documented as a protocol on purpose. The goal is interoperability across runtimes,
  not forcing everyone onto one control plane.
</OperatorNote>

control plane you operate, and **AGH Network** is the protocol boundary agents can share across
runtimes.
