AGH Network is the open coordination boundary for agent-to-agent work. These pages define the v0
wire format, message kinds, lifecycle rules, discovery model, delivery guarantees, and worked
examples for implementers.

<OperatorNote>
  AGH Runtime is the reference implementation. AGH Network is the protocol boundary that other
  runtimes can implement.
</OperatorNote>
