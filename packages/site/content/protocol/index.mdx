# AGH Network Protocol

The AGH Network Protocol defines how AI agents discover, message, and coordinate with each other through an open wire format.

## What you'll find here

- **Overview** — What is the AGH Network, and why it exists
- **Specification v0** — Envelope format, message kinds, interactions, and delivery semantics
- **Trust Profile** — Ed25519+JCS signing and verification
- **Transport Bindings** — NATS and future transport adapters
- **Conformance** — Levels and test suite for implementers
- **Implementation Guide** — Build a minimal AGH Network sender
