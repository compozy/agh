## agh extension status

Show extension runtime status

```
agh extension status <name> [flags]
```

### Options

```
  -h, --help   help for status
```

### Options inherited from parent commands

```
  -o, --output string   Output format: human, json, or toon (default "human")
```

## Environment Diagnostics

Extension manifests may declare `requires_env`. Status output reports those names and the
`missing_env` subset that is unset or empty for the current AGH process environment. AGH never prints
the environment values.

## Output Formats

Every AGH command supports `-o, --output`:

- `human` for interactive terminal use
- `json` for scripts and other machine-readable consumers
- `toon` for compact agent-readable summaries

Example:

```bash
agh extension status <name> -o json
```
