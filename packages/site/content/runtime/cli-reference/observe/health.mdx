## agh observe health

Show observability health

```
agh observe health [flags]
```

### Options

```
  -h, --help   help for health
```

### Options inherited from parent commands

```
  -o, --output string   Output format: human, json, or toon (default "human")
```

## Output Formats

Every AGH command supports `-o, --output`:

- `human` for interactive terminal use
- `json` for scripts and other machine-readable consumers
- `toon` for compact agent-readable summaries

Example:

```bash
agh observe health -o json
```

The JSON payload includes active runtime supervision state in `activities` when sessions have an
active prompt turn. Each activity row reports the session ID, turn ID, last activity age, current
tool when known, stall state, and whether the session is healthy, warning, or stalled.

Use this when watching long-running sessions from automation:

```bash
agh observe health -o json | jq '.activities[] | {session_id, status, current_tool, idle_seconds}'
```
