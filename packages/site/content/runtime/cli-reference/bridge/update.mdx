## agh bridge update

Update mutable bridge fields

```
agh bridge update <id> [flags]
```

### Options

```
      --delivery-defaults string   JSON object or null for delivery target defaults
      --display-name string        New operator-facing bridge display name
  -h, --help                       help for update
      --include-group              Override whether routing includes group identity
      --include-peer               Override whether routing includes peer identity
      --include-thread             Override whether routing includes thread identity
```

### Options inherited from parent commands

```
  -o, --output string   Output format: human, json, jsonl, or toon (default "human")
```

## Output Formats

Every AGH command supports `-o, --output`:

- `human` for interactive terminal use
- `json` for scripts and other machine-readable consumers
- `jsonl` for wait or streaming commands that emit one JSON record per line
- `toon` for compact agent-readable summaries

Example:

```bash
agh bridge update <id> -o json
```
