## agh automation jobs get

Show one automation job

```
agh automation jobs get <id> [flags]
```

### Options

```
  -h, --help   help for get
```

### Options inherited from parent commands

```
  -o, --output string   Output format: human, json, or toon (default "human")
```

## Output Formats

Every AGH command supports `-o, --output`:

- `human` for interactive terminal use
- `json` for scripts and other machine-readable consumers
- `toon` for compact agent-readable summaries

Example:

```bash
agh automation jobs get <id> -o json
```

The job detail output includes durable scheduler metadata when available:

- `next_run` / `scheduler.next_run_at` show the next cursor AGH will wait on
- `scheduler.last_scheduled_at` and `scheduler.last_fire_id` identify the last claimed scheduled fire
- `scheduler.catch_up_policy` is currently `skip_missed`
- `scheduler.misfire_count` counts missed fires reconciled on boot
