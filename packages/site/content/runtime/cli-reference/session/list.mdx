## agh session list

List sessions

```
agh session list [flags]
```

### Examples

```
  # List active sessions
  agh session list

  # Include stopped sessions and filter to a workspace
  agh session list --all --workspace checkout-api
```

### Options

```
      --all                Include stopped sessions
  -h, --help               help for list
      --workspace string   Filter by workspace name or ID
```
