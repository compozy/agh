## agh skill search

Search marketplace skills

```
agh skill search <query> [flags]
```

### Examples

```
  # Search the configured marketplace
  agh skill search "code review"

  # Limit marketplace results
  agh skill search testing --limit 5
```

### Options

```
  -h, --help        help for search
      --limit int   Maximum number of marketplace results to return (default 20)
```

### Options inherited from parent commands

```
  -o, --output string   Output format: human, json, or toon (default "human")
```
