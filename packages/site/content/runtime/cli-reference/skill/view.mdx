## agh skill view

Read a skill or one of its resource files

```
agh skill view <name> [flags]
```

### Examples

```
  # Render a skill as the XML block injected into agents
  agh skill view code-review

  # Read a resource file inside a skill directory
  agh skill view code-review --file references/checklist.md
```

### Options

```
      --file string   Relative file path inside the skill directory
  -h, --help          help for view
```

### Options inherited from parent commands

```
  -o, --output string   Output format: human, json, or toon (default "human")
```
