description: The AGH Runtime operating model, from first install through sessions, memory, automation, extensions, and day-two operations.
---

Core Concepts is the fastest path to understanding how AGH works in practice: install it, launch
durable sessions, learn how replay and memory behave, and then move through the runtime
capabilities that matter once AGH is part of normal work.

<OperatorNote>
  Treat Core Concepts as the place to build operator intuition. It is ordered from mental model, to
  first-run setup, to the runtime systems you manage after AGH becomes part of day-to-day work.
</OperatorNote>
