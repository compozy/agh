Core Concepts is where you build durable operator intuition for AGH. The sections below are grouped
by the stage of adoption you are in, not by internal package layout.

<OperatorNote>
  If you are new, do not read this section front to back. Start with Getting Started, then use the
  other lanes as your work becomes more operational or more advanced.
</OperatorNote>
