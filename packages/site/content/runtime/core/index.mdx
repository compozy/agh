Core is the fastest path to understanding how AGH works in practice: install it, launch durable
sessions, learn how replay and memory behave, and understand the operating model behind the
runtime.

<OperatorNote>
  Treat Core as the place to build operator intuition. It is intentionally ordered from mental
  model, to first-run setup, to the concepts and settings that matter once AGH is part of normal
  work.
</OperatorNote>
