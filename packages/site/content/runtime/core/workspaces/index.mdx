---
title: Workspaces
description: How AGH resolves workspace roots, layers config overlays, and attaches additional roots for monorepos and cross-project agent sharing.
---

A workspace is a registered project root. AGH uses it as the source of truth for workspace-scoped
agents, skills, memory, and config overlays. Sessions inherit the workspace context, and resolution
is deterministic — the registered root wins over global defaults.
