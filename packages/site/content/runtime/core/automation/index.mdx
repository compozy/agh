---
title: Automation
description: Run AGH agents on schedules, in response to runtime events, or from signed external HTTP requests.
---

Automation is the unattended-run surface around three primitives: schedules that fire on a clock,
triggers that react to runtime events, and webhooks that accept signed HTTP from outside the daemon.
All three resolve to the same dispatcher and produce the same audit trail as interactive sessions.
