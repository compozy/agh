---
title: Hooks
description: Intercept named runtime events, inspect the payload, and emit a JSON patch that AGH applies in a deterministic order.
---

Hooks are the policy and observability layer between runtime events and the actions that follow.
A declaration binds a named event to an executor; the executor returns a JSON patch; AGH applies
patches in declaration order and records the outcome on the session event stream.
