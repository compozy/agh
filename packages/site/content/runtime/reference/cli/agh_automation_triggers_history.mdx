## agh automation triggers history

Show run history for one automation trigger

```
agh automation triggers history <id> [flags]
```

### Options

```
  -h, --help            help for history
      --last int        Show only the most recent N runs
      --since string    Show runs since an RFC3339 timestamp or relative duration
      --status string   Filter by run status
      --until string    Show runs until an RFC3339 timestamp or relative duration
```
