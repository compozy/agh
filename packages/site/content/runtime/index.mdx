AGH Runtime is the local-first control plane for real agent work. It lets you run Claude Code,
Codex, Gemini CLI, and other agent CLIs as durable sessions with replayable history,
workspace-aware memory, automation, and observability.

<OperatorNote>
  Runtime docs are organized by operator job, not by internal package boundary. Start in Core to
  understand the operating model, then jump to the CLI or API surface when you need exact command
  or payload detail.
</OperatorNote>

## Working model

- **Core** explains how AGH behaves as a runtime you operate.
- **CLI Reference** tells you exactly what the command surface looks like.
- **API Reference** mirrors the machine-facing contract behind the same control plane.

The intended flow is simple: learn the model once, then use whichever surface fits the job.
