AGH Runtime is the local-first control plane for real agent work. It lets you run Claude Code,
Codex, Gemini CLI, and other agent CLIs as durable sessions with replayable history,
workspace-aware memory, automation, and observability.

<OperatorNote>
  Runtime docs are organized by operator job, not by internal package boundary. Start in Core
  Concepts for the operating model, then use CLI Reference or API Reference when you need the exact
  live contract for one surface.
</OperatorNote>

## Working model

- **Core Concepts** explains how AGH behaves as a runtime you operate.
- **CLI Reference** tells you exactly what the command surface looks like.
- **API Reference** mirrors the machine-facing contract behind the same control plane.

The intended flow is simple: learn the model once in Core Concepts, then use whichever runtime
surface fits the job.
