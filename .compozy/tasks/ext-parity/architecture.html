AGH — Shared Extensibility Resource Runtime

Desired-state control plane for hooks, tools, agents, MCP, skills, automation, bridges, bundles

Write Path (desired state only)

  • • Caller boundary derives MutationActor
  • • CapabilityChecker grants intersection
  • • KindValidator rejects invalid specs
  • • Daemon stamps owner / source / scope
  • • CAS via ExpectedVersion (409 on conflict)
  • • ApplySourceSnapshot serialized per source
  • • Rate limited per actor & per source (429)

Reconcile Pattern

  • • Driver triggered after boot and post-commit
  • • Single-flight per kind, coalesced reruns
  • • Topology via Projector.DependsOn()
  • • Projector.Build reads full snapshot off-path
  • • Projector.Apply swaps atomically
  • • Prior runtime state preserved on failure
  • • No event bus, no public Watch contract

Boundary: Desired vs Operational

  • • resource_records = canonical desired state
  • • resource_source_state = snapshot sessions
  • • Runs (hooks / automation) stay in domain DBs
  • • Bridges: routes & health stay in domain
  • • Bundles: composition owned via owner index
  • • Operational APIs remain typed & domain-specific
  • • Extension manager loses catalog authority
