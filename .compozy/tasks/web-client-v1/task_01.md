# Task 01: Foundation — Dependencies, Types & Layout Shell

## Overview

Set up the project foundation: install all required NPM dependencies, create TypeScript type definitions for the 3 domain systems (session, agent, daemon), create barrel export files, and build the app layout shell with TanStack Router routes including sidebar provider and theme support. This task establishes every shared type and route that downstream tasks depend on.

- REFERENCE TECHSPEC "Routing Structure" section for route layout

- MINIMIZE CODE — show code only to illustrate current structure or problem areas

</critical>

<requirements>
- MUST install all NPM dependencies: `@ai-sdk/react`, `ai`, `@tanstack/react-virtual`, `react-markdown`, `remark-gfm`, `react-syntax-highlighter`, `@types/react-syntax-highlighter`
- MUST create TypeScript interfaces matching the daemon HTTP API response shapes (see TechSpec "Data Models" section)
- MUST create `UIMessage`, `ToolUseResult`, `SessionPayload`, `AgentPayload`, `HealthPayload`, `PermissionRequest`, `AgentEventPayload`, `TokenUsagePayload`, `TurnHistoryPayload`, `SessionEventPayload` types
- MUST define API response envelope types matching actual backend responses: `{ sessions: SessionPayload[] }`, `{ session: SessionPayload }`, `{ events: SessionEventPayload[] }`, `{ history: TurnHistoryPayload[] }`, `{ agents: AgentPayload[] }`, `{ health: HealthPayload }`
