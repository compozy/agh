# Per-Package Improvements Pass — Task List

One task per package under `internal/*`. Each task runs the five-skill pipeline (`refactoring-analysis`, `extreme-software-optimization`, `ubs`, `deadlock-finder-and-fixer`, `security-review`) against a single package, lands fixes, and writes a report at `.compozy/tasks/improvs/reports/<package>.md`. Tasks are independent and can run in any order.
