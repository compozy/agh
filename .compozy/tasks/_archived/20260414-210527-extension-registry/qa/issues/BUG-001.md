# BUG-001: Local extension remove deleted the original source directory

**Severity:** Critical
**Priority:** P0
**Type:** Data
**Status:** Fixed

## Environment

- **Build:** `9cb1a73-dirty`
- **OS:** macOS
- **Browser:** not applicable
- **URL:** CLI `agh extension install/remove` and daemon extension service

## Summary

Installing a local extension from a repository path persisted that original source path as the installed manifest location. Removing the extension later deleted the repository directory instead of a managed install copy, causing real source-file loss in the worktree.

## Reproduction

```bash
AGH_HOME=/tmp/agh-ext-reg ./bin/agh extension install sdk/examples/secret-guard --output json
AGH_HOME=/tmp/agh-ext-reg ./bin/agh extension remove secret-guard --output json
test -e sdk/examples/secret-guard/extension.toml && echo EXISTS || echo MISSING
```

Observed before the fix:

- `extension remove` reported the repository path as the removed directory.
- `sdk/examples/secret-guard/extension.toml` and the rest of the source tree were deleted from the worktree.

## Expected

Local installs must be copied into `AGH_HOME/extensions/<name>` before registry persistence, and removing the extension must delete only that managed copy.

## Root cause

Both offline CLI installs and daemon-backed installs called `registry.Install(...)` with the original local source directory path. The registry therefore persisted the source directory as `manifest_path`, and the remove flow trusted `filepath.Dir(info.ManifestPath)` as the on-disk install location.

## Fix

Added a shared managed-install helper in `internal/extension/install_managed.go` that stages and copies local extensions into `AGH_HOME/extensions/<name>` before persisting registry metadata. Updated both offline CLI installs and daemon extension installs to use that helper.

## Verification

- `go test ./internal/cli -count=1 -run 'TestInstallPreparedExtensionDetectsChecksumMismatch|TestExtensionInstallAndRemoveOfflinePreservesSourceDirectory|TestExtensionInstallCommandInstallsMarketplaceExtensionAndPrintsRestartMessage|TestExtensionRemoveCommandDeletesDirectoryAndRegistryRecord|TestExtensionUpdateCommandReinstallsNewerVersion'`
- `go test ./internal/daemon -count=1 -run 'TestBootExtensionsLogsStartFailureAndKeepsPartialRuntime|TestDaemonExtensionServiceInstallStatusAndDisable|TestDaemonTestExtensionManifest'`
- `make verify`
- Manual isolated daemon replay using `sdk/examples/secret-guard`, confirming install persisted `/tmp/agh-ext-reg/extensions/secret-guard/extension.toml` and remove left `sdk/examples/secret-guard/extension.toml` intact.

## Impact

- **Users Affected:** all users installing local extensions from workspace paths
- **Frequency:** always
- **Workaround:** none
