## Summary by CodeRabbit

- **New Features**
  - CLI marketplace: search, install, remove, and update skills from a registry (default)
  - Skill lifecycle hooks that run on session created/stopped events
  - Skills may declare MCP servers (consent-controlled inclusion)
  - Provenance & integrity checks: installed skills include sidecars and hash verification
  - Config: marketplace registry and optional base URL + marketplace consent lists

- **Tests**
  - Extensive integration and unit tests covering marketplace flows, hooks, and provenance

## Walkthrough

Adds marketplace-backed skill lifecycle (search/install/remove/update) to the CLI, provenance sidecars & integrity checks for installed skills, MCP server declarations and resolver, subprocess-based skill hooks executed during session lifecycle, and wiring of these components into daemon/session notifier and manager flows.
